English practice helper

Description

I'm learning english and this is my idea of website which can help improve learning process in some things I'm (or maybe also others) struggling with.

At this version of website user can get some random numbers, times, dates and hint for roman numerals months names.

Idea of usage

If you are struggling with numbers, times and dates in english just generate them by clicking on buttons and read it on voice. After some days of practice you will see progress.
